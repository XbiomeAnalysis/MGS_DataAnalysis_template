6.1 PERMANOVA

metaphlan2_ps_per <- run_permanova(ps=metaphlan2_ps_species, method = "bray", columns = "Group")
print(metaphlan2_ps_per)

6.3 Ordination

We performed ordination by using Principal Coordinate Analysis (PCoA).

metaphlan2_ps_ordination <- run_ordination(
                       ps = metaphlan2_ps_species,
                       group = "Group",
                       method = "PCoA")
